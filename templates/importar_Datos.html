Protocolo de Importación

Siga estas directrices para asegurar la integridad de la base de datos de la licorera:

  • Formato requerido: .xlsx (Excel).
  • La hoja de cálculo debe llamarse exactamente: "Producto".
  • ADVERTENCIA: Esta acción reemplaza todo el inventario actual.

Estructura de Columnas:

Encabezado Estado Descripción
nombre REQUERIDO Nombre del licor o insumo.
valor_venta REQUERIDO Precio de venta al público.
cantidad Opcional Stock disponible (Default: 0).
codigo Opcional Código de barras (Auto-generado si falta).

Subir Archivo

Verifique los datos antes de procesar. El sistema realizará un "Wipe" de los productos actuales.
